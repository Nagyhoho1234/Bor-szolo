# Addressing the Implementation Problem in Agricultural Decision Support Systems: The Example of vite.net

- **Authors:** Vittorio Rossi, Francesca Salinari, et al.
- **Year:** 2014
- **Journal:** Computers and Electronics in Agriculture, 100, 88-99
- **DOI:** (ScienceDirect)
- **Access:** [ScienceDirect](https://www.sciencedirect.com/science/article/abs/pii/S0168169913002536) (paywalled); [Academia](https://www.academia.edu/92468521)
- **Status:** PAYWALLED - abstract available

## Abstract

Presents the vite.net decision support system for grapevine farmers, integrating monitoring of basic vineyard parameters (air, soil, plants, pests, diseases) with software tools that analyze data and provide alerts.

## Key Features

- Monitors air temperature, humidity, soil conditions, plant status, pest/disease pressure
- Integrates disease models (including downy mildew, powdery mildew, Botrytis)
- Provides pesticide treatment timing recommendations
- Addresses the common "implementation problem" of DSS adoption by farmers

## Relevance to Tokaj Research

Demonstrates how station-based disease models can be operationalized into practical decision support. The 50-station network could feed a similar DSS for Tokaj growers.
