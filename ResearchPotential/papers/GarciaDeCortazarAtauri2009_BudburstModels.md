# Performance of Several Models for Predicting Budburst Date of Grapevine (Vitis vinifera L.)

- **Authors:** Inaki Garcia de Cortazar-Atauri, Nadine Brisson, Jean-Pierre Gaudillere
- **Year:** 2009
- **Journal:** International Journal of Biometeorology, 53, 317-326
- **DOI:** 10.1007/s00484-009-0217-4
- **Access:** [Springer Link](https://link.springer.com/article/10.1007/s00484-009-0217-4) (paywalled); [ResearchGate](https://www.researchgate.net/publication/24195173)
- **Status:** PAYWALLED - abstract available

## Abstract

Compared several phenological models for predicting grapevine budburst date. The BRIN model showed better performance than previous models, with improvements explained by calculation of dormancy, use of hourly temperatures, and base temperature. Base temperature was the main driver, while dormancy period was not significant.

## Key Findings

- BRIN model outperforms simpler forcing-only models for budburst prediction
- Base temperature is the most important parameter
- Dormancy calculation adds relatively little improvement
- Hourly temperature resolution improves predictions vs daily data
- Important for understanding the beginning of the growing season

## Relevance to Tokaj Research

Provides context for phenology modeling (Direction 4). If sub-daily temperature data is available from the 50-station network, the BRIN model may outperform the simpler GFV model for budburst prediction.
