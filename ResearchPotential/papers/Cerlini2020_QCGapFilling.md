# Quality Control and Gap-Filling Methods Applied to Hourly Temperature Observations over Central Italy

- **Authors:** Paolina B. Cerlini, Lorenzo Silvestri, Miriam Saraceni
- **Year:** 2020
- **Journal:** Meteorological Applications, 27(3), e1913
- **DOI:** 10.1002/met.1913
- **Access:** [Wiley](https://rmets.onlinelibrary.wiley.com/doi/10.1002/met.1913)
- **Status:** ACCESSIBLE at Wiley

## Abstract

Analyzed hourly temperature time series from 2010-2017 in the Umbria region of central Italy, applying WMO-standard quality control procedures. Validation consisted of automatic QC producing validated data with metadata in NetCDF format, followed by manual checks and EOF-based reconstruction.

## Key Findings

- Two-step validation-reconstruction procedure following WMO standards
- Linear interpolation for 1-hour gaps; EOF algorithm for gaps >= 2 hours
- 7 stations failed validation (<80% valid data); 67 stations suitable for reconstruction
- ERA5 provides independent gap-filling source

## Relevance to Tokaj Research

Provides a WMO-compliant QC and gap-filling workflow applicable to the 50-station network. The two-step approach (validation then reconstruction) and the 80% valid data threshold are practical guidelines.
