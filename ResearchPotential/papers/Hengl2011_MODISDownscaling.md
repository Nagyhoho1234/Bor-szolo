## Summary (from research plan context)

Achieved R2 = 0.84 using local space-time regressions to statistically downscale MODIS LST (1 km) using station data and DEM covariates for daily thermal information.

## Key Contribution

- Demonstrated effective statistical downscaling of MODIS LST
- Local space-time regression approach
- Combined satellite thermal data with ground station measurements
- DEM covariates improve downscaling accuracy
